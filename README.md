A lightweight client-side Fabric mod that automatically switches your hotbar slot after placing blocks.

The Lite version keeps only the core functionality with a single built-in **Smart** mode and no configuration system.

Designed for players who want a simple plug-and-play experience with maximum compatibility and faster updates for new Minecraft versions.

## Features

* Smart automatic hotbar block switching
* Prevents repetitive block placement patterns
* Lightweight and minimal
* No configs or setup required
* Overlay icon near crosshair
* Client-side only

## Default Keybind

* Press **R** to enable/disable Random Placement mode.
